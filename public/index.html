Xorb Proxy

Secure CORS Proxy by Xorbious

GitHub
Important: This CORS proxy is designed to help you bypass cross-origin restrictions when making API requests.

1. Basic Usage

To use the CORS proxy, send requests to /proxy endpoint with the target URL as a query parameter.

Example Request

https://xorb-proxy.xorbious.com/proxy?url=https://api.example.com/data

2. Supported HTTP Methods

  • GET
  • POST
  • PUT
  • DELETE
  • OPTIONS

3. Query Parameters

Parameter Description Required
url Full URL of the target API endpoint ✓ Yes
Additional Parameters Any extra query parameters for the target URL Optional

4. Supported Headers

by default the proxy will try to access the following headers if they are present in the request headers

  • Content-Type
  • Authorization
  • Accept
  • UserAgent
  • Refferer
  • X-Requested-With

Incase you want the proxy, to access some extra headers from the request header, you can pass in an extra query params called extraHeaders, with the key of the headers you want comma delimitted. see example

4. Code Examples

JavaScript Fetch Example

fetch('https://xorb-proxy.xorbious.com/proxy?url=https://api.example.com/users', {
    method: 'GET',
    headers: {
        'Content-Type': 'application/json'
    }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
Axios Example

axios.get('https://xorb-proxy.xorbious.com/proxy', {
    params: {
        url: 'https://api.example.com/data',
        extraParam: 'value'
    }
})
.then(response => console.log(response.data))
.catch(error => console.error('Error:', error));
Extra Header Example

axios.get('https://xorb-proxy.xorbious.com/proxy', {
    params: {
        url: 'https://api.example.com/data',
        extraParam: 'value',
        extraHeaders:"x-custom-header1,x-custom-header2,Cookie" // case insensitive
    },
    headers:{
        x-custom-header1:"a custom header value 1",
        x-custom-header2:"a custom header value 2",
        //or any other not specified in the defaults
        Cookie: "name=value; name2=value2; name3=value3"
    }
})
.then(response => console.log(response.data))
.catch(error => console.error('Error:', error));

5. Limitations & Restrictions

  • Maximum request body size: 100KB
  • Rate limited to 100 requests per 15 minutes per IP
  • Only HTTPS URLs are recommended

6. Error Handling

Status Code Description
400 Invalid or missing URL
403 Origin not allowed (if restrictions are enabled)
500 Internal server error
504 Gateway timeout
